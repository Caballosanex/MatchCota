# Terraform Infrastructure

Aquesta carpeta contindrà tota la configuració IaC per desplegar MatchCota a AWS.

## Estructura

- `modules/` - Mòduls reutilitzables (networking, compute, database, storage, security)
- `environments/dev/` - Configuració per entorn de desenvolupament
- `environments/prod/` - Configuració per entorn de producció
